В программе не корректно обозначен диапазон возраста тинэйджеров(<19).
Должно быть от 13 включительно и до 19 включительно(>= 13 && age <= 19).
public class Person {
    public static boolean isTeenager(int age)
    {
        boolean result;
        if (age >= 13 && age <= 19) {
            result = true;
        }
        else {
            result = false;
        }
        return result;
    }
}
